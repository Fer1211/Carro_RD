# CarroRD

Sitio web para publicar y vender vehículos en República Dominicana.

## Estructura del proyecto
- `index.html`: HTML principal
- `css/`: Estilos CSS
- `js/`: Scripts JavaScript
- `img/`: Imágenes y recursos

## Uso
1. Edita la configuración en el archivo JS según tus necesidades.
2. Sube el contenido a tu servidor o GitHub Pages.
